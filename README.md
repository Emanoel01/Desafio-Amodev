# Desafio-Amodev
Projeto feito em Next.JS para conclusão de um desafio

# Objetvo:

# Para compilar o projeto:
  - Clonar o repositório;
  - Pelo CMD, entrar na raiz do projeto;
  - digitar o comando yarn run dev
    - obs: pode ser pelo npm também
